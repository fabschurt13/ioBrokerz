## Адаптер miio для ioBroker
Этот адаптер управляет устройствами, которые следуют протоколу miIO.

## Текущий поддерживаемый тип устройства
- свежий воздух
- воздушный монитор
- очиститель воздуха
- поклонник
- увлажнитель
- чайник
- свет
- вилка
- удлинитель
- Waterpuri
