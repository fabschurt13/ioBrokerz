### Ссылка
Этот виджет соответствует виджету "HTML-фрейм", но также является интерактивной ссылкой по всей своей поверхности. Может использоваться для навигации между представлениями или для внешних ссылок.

Атрибут | Описание |
---- | ---- |
html | самоочевидный;) ... вставьте сюда HTML-код для форматированного представления текста.
ссылка | URL ссылки. Чтобы использовать ссылку на другое представление, просто введите имя представления, которому предшествует символ хеша (#) target | Цель ссылки. Оставьте пустым, чтобы остаться в том же окне браузера; хотел бы открыть новое окно _blank. Другие параметры: _self (та же вкладка), _parent (), _top ()

### HTML-навигация
Этот виджет используется для создания навигации между представлениями. Соответствует виджету «статическая ссылка», но используется только для навигации между представлениями и дополнительно предлагает возможность использовать анимированные эффекты при смене видов.

Атрибут | Описание |
---- | ---- |
html | Самоочевидный;) ... вставьте сюда HTML-код nav_view | Здесь вы должны ввести имя представления, по которому вы хотите перейти, к hide_effect | Здесь вы можете ввести имя эффекта jQueryUI, который будет использоваться при выходе из представления. Доступны следующие эффекты: «Слепой», «Отскок», «Скрепление», «Падение», «Взрыв», «Исчезновение», «Сгиб», «Выделение», «Слойка», «Пульсация», «Масштаб», «Встряхивание», «Размер», «Слайд» и «Перенос».
hide_duration | Длительность эффекта в мс. show_effect | см. выше, то же самое - но на этот раз для отображения нового представления. show_duration | См. выше, время в мс для отображения нового представления.
