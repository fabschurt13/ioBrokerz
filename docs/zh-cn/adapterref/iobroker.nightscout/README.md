适用于ioBroker的## nightscout适配器
适配器从Nighscout页面获取数据。 Nightscout（云计算中的CGM）是一个开源的DIY项目，允许通过个人网站，智能手表查看器或智能手机可用的应用程序和小部件实时访问CGM数据。
