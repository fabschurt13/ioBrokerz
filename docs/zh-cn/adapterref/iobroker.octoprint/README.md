＃ioBroker.octoprint
将OctoPrint连接到ioBroker的适配器

＃＃ 特征
-获取版本信息
-获取打印机信息
-获取当前的打印作业信息
-设定工具温度
-设定床温
-通过命令连接，断开连接和家用打印机
