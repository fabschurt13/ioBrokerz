##用于读取iOBroker的Kostal Piko BA数据的适配器
用于读取Kostal Piko BA数据的适配器。适配器创建一些状态并顺序更新。

###设置
要连接到Kostal Pico BA逆变器，请在配置中输入其IP地址。
