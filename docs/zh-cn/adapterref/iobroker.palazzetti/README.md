## IoBroker的palazzetti适配器
__控制[palazzetti颗粒炉](https://palazzettigroup.com/products/pellet-stoves/)连接到接线盒.__

__对象通道：__

*控制：控制炉灶的属性
*获取：当前炉灶数据
* info：设备信息
*计时器：计时器设置
