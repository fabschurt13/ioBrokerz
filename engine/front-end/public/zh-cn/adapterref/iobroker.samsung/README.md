### IoBroker.samsung
####说明
三星电视适配器

####配置
输入Samsung TV的IP

####安装
通过ioBroker管理员。

Otherweise在iobroker根目录中执行以下命令（例如在/ opt / iobroker中）

```
iobroker install samsung
```

要么

```
npm install iobroker.samsung
```

＃＃＃＃ 要求
三星电视2014年之前<br>自2016年起支持设备（版本0.2.0，实验）

安装后，您必须确认电视上的新连接
